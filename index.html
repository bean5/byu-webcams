BYU Webcams

Google Chrome extension that displays many BYU Webcams and updates them often (sometimes every second). See website for non-extension web version.

Welcome to BYU Webcams

Brigham Young University has some great public webcams, but finding them can be difficult. So, I built this Google Chrome extension that you can use with one click. All you have to do is install this free Google Chrome app by going to https://chrome.google.com/webstore/detail/byu-webcams/popkdjdpaicpdnfpgopjkmhibkngomho.

Optionally, you may install the app from the source code by downloading it above or by cloning it at Github at https://github.com/bean5/byu-webcams.
